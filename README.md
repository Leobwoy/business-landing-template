# Nexus — IT Networking & AWS Cloud Solutions

A modern, highly-interactive landing page built for the **Nexus** IT networking team project. It showcases our AWS Cloud infrastructure, threat analysis tooling, and team members. Built with pure HTML, CSS, and vanilla JavaScript for maximum performance and a customized "Cyber/Cloud" aesthetic.

## ✨ Features

- **Dynamic Hero Section** — Full-screen hero with live animated stats counters.
- **Live Dashboard Mockup** — Simulated server log terminal (`INFO`, `WARN`, `CRIT`) and animated "Threats Blocked" bar charts.
- **AWS Architecture Flowchart** — A pure CSS and HTML flowchart demonstrating our infrastructure (AWS WAF -> GuardDuty -> EC2 / S3) with glowing, animated data packets.
- **Team Roster** — Profiles of our team members with professional headshots.
- **Interactive Contact Form** — A presentation-ready contact form simulation that displays a loading spinner and success message without refreshing the page.
- **Scroll Reveal Animations** — Vanilla JavaScript `IntersectionObserver` smoothly fades and slides elements into view as you scroll.
- **Cyber/Cloud Theme** — A vibrant color palette featuring deep navy blues (`#040914`) and glowing neon cyan accents (`#00f2fe`).
- **Fully Responsive** — Beautifully scales from mobile to ultra-wide desktop monitors.

## 🚀 Quick Start

1. **Clone** this repository:
   ```bash
   git clone https://github.com/Leobwoy/business-landing-template.git
   ```
2. Navigate into the project directory:
   ```bash
   cd business-landing-template
   ```
3. Open `index.html` in any modern web browser or start a local HTTP server:
   ```bash
   python -m http.server 8080
   ```
4. View the site at `http://localhost:8080`.

## 🛠️ Tech Stack

- **HTML5**: Semantic markup.
- **CSS3**: Custom properties (variables), Grid, Flexbox, keyframe animations, and custom scrollbars.
- **JavaScript (Vanilla)**: DOM manipulation, event listeners, and `IntersectionObserver` API for scroll-linked animations.
- **Font Awesome**: Used for scalable vector icons throughout the UI.
- **Google Fonts**: Inter font family for sleek typography.
